# Minikube Setup and Namespace Creation

## Objective
To set up Minikube locally and explore Kubernetes namespaces.

## Tools Used
- Docker Desktop
- Minikube
- kubectl
- Windows OS

## Steps Performed
1. Installed Minikube and verified version
2. Started Minikube using Docker driver
3. Created a namespace called dev
4. Deployed an nginx pod inside dev namespace
5. Verified pods across namespaces

## Conclusion
Minikube was successfully set up and namespaces were explored.
